# SECURITY POLICY

## Scope
This repository is a dual-use research and advocacy artifact. It is published to document and demonstrate a systemic vulnerability in academic peer review and related evaluative workflows.

Certain project behaviors, including provenance and grounding markers in generated outputs, are intentional safety features designed to reduce the risk that outputs are mistaken for authentic human-authored reviews, to preserve local grounding, and to deter prohibited evaluative use.

The authors did not intend to do harm, do not authorize harmful use, and do not authorize use of this repository or any output derived from it in any real-world peer review, editorial review, program committee, grant, hiring, promotion, moderation, adjudication, or other evaluative workflow.

## Local Processing / No Maintainer Receipt
The project is intended to operate locally. The maintainers do not request, collect, receive, or monitor provenance or grounding markers through the project itself. Any third-party disclosure of an output occurs only if a user or downstream actor chooses to copy, submit, publish, or otherwise share that output.

## Abuse Reporting
If you believe this repository or any derivative output has been used in a real-world evaluative context, or in a way that creates privacy, provenance, or security concerns, please contact the maintainers with the following information:

Please include:
- a link or copy of the relevant material
- a short description of the suspected misuse
- any evidence of third-party submission, publication, or reliance

## Maintainer Position
The maintainers will not assist, optimize, troubleshoot, or adapt this project for any real-world evaluative use.
Where credible misuse is reported, maintainers may modify content, add further warnings, restrict examples, or remove materials at their discretion.
